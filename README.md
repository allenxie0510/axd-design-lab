# AXD Design Lab

一个专业的品牌设计工作室网站，采用 frog 设计风格。

## 特点

- **多页面网站**：首页、作品列表、作品详情、后台管理
- **后台管理功能**：添加、编辑、删除作品
- **响应式设计**：适配桌面和移动设备
- ** frog 设计风格**：专业、权威、黑白对比

## 页面

- `/` - 首页（作品展示、服务、流程、关于、联系）
- `/works` - 作品列表
- `/works/[id]` - 作品详情
- `/admin` - 后台管理

## 技术栈

- Next.js 14
- React + TypeScript
- Tailwind CSS
- 静态导出

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建输出在 `out` 文件夹中。
